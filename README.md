# 📊 E-Commerce Dashboard Analysis

## 📌 Project Overview
This project analyzes sales performance across multiple marketplaces:
- Shopee
- Tokopedia
- Lazada
- Blibli

The dashboard provides a clear view of revenue trends, product performance, and refund behavior to support data-driven business decisions.

---

## 🎯 Objectives
- Analyze total revenue and growth
- Compare marketplace performance
- Identify top-selling product categories
- Monitor refund rate and customer behavior

---

## 📈 Key Metrics
- Total Revenue
- Total Quantity Sold
- Average Order Value (AOV)
- Refund Rate

---

## 🛠 Tools Used
- Google Looker Studio (Dashboard Visualization)
- Microsoft Excel (Data Processing)
- Data Cleaning & Transformation

---

## 🔥 Key Insights
- Shopee contributes the highest revenue share among all marketplaces
- Total quantity shows consistent growth (+6.8%)
- Refund rate is relatively low (~1.3%), indicating healthy transactions
- Formal category dominates overall sales performance
